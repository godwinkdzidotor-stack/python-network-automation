Firewall Rule Auditor
Rule Hygiene
  • Takes firewall rule exports (CSV or similar).
  • Flags overly permissive patterns like src=any, dst=any, or 0.0.0.0/0.
  • Helps quickly identify rules that need tightening.

Subnet Reachability Scanner
Visibility
  • Performs ICMP ping sweeps across a provided CIDR.
  • Reports which IPs are responsive vs. unreachable.
  • Useful for quick inventory and lab validation.

Example Use Cases
How It Helps
  • Firewall hygiene review: run the auditor before major audits to quickly surface risky rules.
  • Lab design checks: after building a new topology, scan subnets to confirm devices are reachable.
  • Training & demos: use scripts to teach junior engineers about network security automation.
